# SEO TODO: OG Image

Je moet nog een Open Graph image maken voor social media previews.

## Specificaties:
- **Afmetingen**: 1200 x 630 pixels
- **Format**: JPG of PNG
- **Locatie**: `/public/og-image.jpg`

## Wat erop moet:
1. Je logo/branding
2. Text: "Full-Stack Software Engineer"
3. Text: "BarTech" (of je bedrijfsnaam)
4. Eventueel: Je tagline

## Tools om te gebruiken:
- **Canva** (gratis): https://www.canva.com/
  - Zoek template: "Open Graph Image"
  - Customize met je branding
  
- **Figma** (gratis): https://www.figma.com/
  - Maak 1200x630 frame
  - Design je image
  
- **Adobe Express** (gratis): https://www.adobe.com/express/

## Preview Tool:
Na maken, test hoe het eruitziet:
- https://www.opengraph.xyz/
- https://www.bannerbear.com/tools/og-image-preview/

- [ ] Image gemaakt (1200x630px)

- [ ] Getest op preview tool
